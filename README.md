## WhatsAMenu website

This is the website for the WhatsAMenu app. It is built using Vue.js and Vuetify.

It uses the [WhatsAMenu API](https://github.com/wiredmartian/whatsamenu-api-documentation)

### Project setup
```
yarn install
```

#### Compiles and hot-reloads for development
```
yarn run serve
```
